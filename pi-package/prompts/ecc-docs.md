# ECC Docs Update

> Based on ECC's update-docs command

Focus: ${1:-Audit project for documentation gaps}

## Documentation Checklist

- [ ] **README.md** — Is the project description, setup, and usage current?
- [ ] **API docs** — Are all endpoints/routes documented?
- [ ] **Architecture docs** — Do component diagrams match reality?
- [ ] **CHANGELOG.md** — Are recent changes recorded?
- [ ] **Comments** — Do inline comments still match the code?

## Process
1. Read the current docs
2. Compare with actual code
3. Update outdated information
4. Add missing documentation
5. Remove obsolete content
